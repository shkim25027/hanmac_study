1. Gulp 설치

   1-1 .Gulp CLI 전역 설치(권장)
   npm install --global gulp-cli

   1-2. Gulp 프로젝트 설치(로컬)
   npm install --save-dev gulp

2. Gulp 실행

   터미널에서 프로젝트 루트에서: gulp

3. Gulp dist 경로 수정은 gulpfile.js 파일

scss 폴더 정리 내용
Abstracts: 변수, 함수, mixins (CSS 출력 없음)
Base: 리셋, 타이포그래피, 애니메이션
Components: 재사용 가능한 UI 컴포넌트
Layout: 주요 레이아웃 구조
Utilities: 헬퍼 클래스
