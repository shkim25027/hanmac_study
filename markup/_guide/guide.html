8. 주요 페이지별 스타일 가이드

메인 페이지 (index.html)

  • 배경: 그라데이션 원형 요소 (bg-circle) 사용
  • 비디오 카드: 360px × 144px, border-radius: 120px
  • 키워드 영역: 상단 중앙 배치, 허용/차단 태그 구분
  • 게이지 차트: SVG 기반 원형 게이지

학습 페이지 (learning.html)

  • 학습 게이지: SVG 기반 진행률 표시
  • 챕터 카드: 200px 너비, 상태별 색상 구분 (base/current/completed)
  • 진행 상태: 진행중(#ff5c00), 완료(#008c49), 미시작(#8d8888)
  • 마커: 학습 지점 표시용 SVG 마커

온보딩 페이지 (onboarding.html)

  • 퍼즐 보드: SVG 기반 인터랙티브 퍼즐
  • 퍼즐 조각: 4개 그룹 (가족사, 한맥가족의 DX, 가치공유, 회사생활)
  • 각 그룹별 색상: #7ed29b, #ffccca, #87a7d6, #b49a65

인트로 페이지 (intro.html)

  • 카드 리스트: 3개 카드, 465px 너비
  • 카드 스타일: 흰색 배경, border-radius: 0 0 12px 12px
  • 애니메이션: 페이드인 + 슬라이드업 효과
  • CTA 버튼: 하단 중앙 배치
