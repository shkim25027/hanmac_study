# 마이클래스 페이지(myclass_02) UI 구현 가이드

참조 이미지 기준으로 현재 페이지를 동일하게 만들기 위한 가이드입니다.

---

## 1. 전체 레이아웃 구조

```
┌─────────────────────────────────────────────────────────────────┐
│  헤더 (배움터 로고, 마이클래스/온보딩/법정교육…, 검색/알림/프로필)  │
├─────────────────────────────────────────────────────────────────┤
│  사용자 정보 바: "홍길동님 현재 128등 (지난 주 대비 44↑)" [▼]      │
├─────────────────────────────────────────────────────────────────┤
│  페이지 인트로 (목표 선택 여부에 따라 문구 변경)                    │
├──────────┬──────────────────────────────────────────────────────┤
│  연도/   │  책장 (학습 목표 카드들이 책 형태로 나열)               │
│  분기    │  - 1행: 책 카드들                                      │
│  선택    │  - 선반                                                │
│  목표    │  - 2행: 책 카드들                                      │
│  상세    │  - 선반                                                │
│  [목표   │  하단: IQ, IT 테크, 웰니스, 마인드셋, 리더십 태그      │
│   다시   │                                                        │
│   선택]  │                                                        │
├──────────┴──────────────────────────────────────────────────────┤
│  [학습목표 고르기] / [추가 목표 고르기]  (CTA)                     │
└─────────────────────────────────────────────────────────────────┘
```

- **목표 미선택**: 인트로는 "학습 목표를 골라볼까요?", 연도 선택만 보이고 좌측 목표 상세 영역 비노출 또는 빈 상태, CTA는 "학습목표 고르기".
- **목표 선택 후**: 인트로는 "[목표명], 지금부터 하나씩 완독해보세요." + D-day 문구, 좌측에 선택 목표 상세 + "목표 다시 선택", 책장 첫 번째가 선택된 책(표지 이미지), CTA는 "추가 목표 고르기".

---

## 2. 영역별 상세 스펙

### 2.1 사용자 정보 바 (user-info-bar)

| 항목 | 현재 코드 | 이미지 기준 수정 |
|------|-----------|------------------|
| 문구 | `홍길동님 현재 1250등` | `홍길동님 현재 128등 (지난 주 대비 44↑)` |
| 표시 | `rank-text` 내 `<em>1250등</em>` | 등수 + 괄호 안에 전주 대비 순위 변동(예: 44↑ / 3↓) |
| 버튼 | `btn-rank-toggle` (순위 상세) | 유지. 클릭 시 순위 상세 토글 |

- **데이터**: 등수, 지난 주 대비 변동값(숫자, 상승/하락)을 서버/상태에서 내려주는 구조로 확장 권장.

---

### 2.2 페이지 인트로 (page-intro)

**상태 1: 목표 미설정 (initial)**

- 문구 1: `홍길동 선임연구원님, **학습 목표**를 골라볼까요?`
- 문구 2: `**D-64(3월 30일)** **까지 선택한 목표**와 함께 배움을 채워가세요.`

**상태 2: 목표 설정 후 (done)** — 이미지와 동일하게

- 문구 1: `**[자기이해와 강점찾기],** 지금부터 하나씩 완독해보세요.`  
  → 선택된 목표명을 **굵은 보라색**으로 강조.
- 문구 2: `D-64(3월 30일) 까지 나만의 지식을 채워갈 시간입니다.`  
  → **주황색** 계열로 표시.

**마크업 권장**

- `intro-state--initial` / `intro-state--done` 유지.
- "목표 설정 후" 문구 1에서 목표명만 동적 치환 (예: `[${goalTitle}], 지금부터 하나씩 완독해보세요.`).
- "목표 설정 후" 문구 2는 `intro-dday` 또는 별도 클래스로 색상 적용.

---

### 2.3 좌측 사이드바 (목표 선택 시에만 노출)

이미지에는 다음이 한 묶음으로 왼쪽에 있습니다.

| 요소 | 스펙 |
|------|------|
| 연도 | `2026년` 드롭다운 (기존 `year-selector` 활용) |
| 분기 | `▶ 1분기` 텍스트 (또는 버튼) |
| 선택 목표 제목 | 예: "자기이해와 강점 찾기" |
| 선택 목표 설명 | 예: "나의 성향, 가치관, 강점 등을 탐색하고..." |
| 버튼 | `← 목표 다시 선택` (클릭 시 목표 선택 해제 또는 다시 고르기 플로우) |

**레이아웃**

- `year-selector`와 한 컬럼으로 묶어서 왼쪽 고정.
- 목표 미선택 시: 연도만 보이거나, 목표 영역은 비움/숨김.
- 목표 선택 시: 연도 + 분기 + 선택 목표(제목, 설명) + "목표 다시 선택" 표시.

**JS**

- `applySelectedState(goalId)` 시 해당 목표의 `title`, `desc`로 좌측 텍스트 갱신.
- "목표 다시 선택" 클릭 시 `selectedGoalId = null`, 인트로/카드/CTA를 미선택 상태로 복원.

---

### 2.4 책장(bookshelf) — 책 카드

**공통**

- 각 카드는 "책" 형태. 선반(shelf) 위에 여러 권 나열.
- 카드마다: **분기 배지**(예: 1분기), **하트 아이콘**(우상단), **제목**, **설명**, 하단 **카테고리 태그**(예: IT 테크, 웰니스).

**선택된 카드 (is-selected)**

- `card-books-area` 표시: **책 표지 이미지** (`Group 1171286226.png` 등).
- 좌상단 **quarter-badge** "1분기".
- 우상단 **하트 아이콘**.
- 제목 예: "조직도가 리셋된다!" (이미지 기준; 실제 데이터는 목표별 추천 도서/코스 제목으로 연동 가능).
- 부제/설명: "위계 중심 조직은 더 이상 통하지 않습니다.", "더 유연하게 일할 방법".
- 하단 태그: "IT 테크".

**미선택 카드**

- `card-default-area` 표시: 아이콘 + 제목 + 설명 (현재 구조 유지).
- 또는 이미지처럼 **책 표지 스타일**을 유지하되, 파스텔 배경 + 빈 표지/썸네일 + 제목/설명/태그만 다르게 표시.
- 예시 제목: "혈당 스파이크", "제대로 쉬는 방법", "'AI 도파민'의 바다에 빠진 이유" 등 — 목표별 추천 콘텐츠 또는 목표 카드용 부제로 매핑.

**카드 색상**

- 선택된 책: 옅은 청록(teal) 계열.
- 미선택: 파스텔톤(베이지, 분홍, 연녹색, 연노랑 등)을 카드별로 지정 가능 (예: `goal-item--color-1` ~ `goal-item--color-n`).

---

### 2.5 책장 하단 태그

- 선반 아래에 태그 목록: **IQ**, **IT 테크**, **웰니스**, **마인드셋**, **리더십**.
- 클릭 시 해당 태그 필터(선택 사항) 또는 시각만 강조해도 됨.
- 마크업 예: `bookshelf` 안 `shelf-row` 다음에 `<div class="shelf-tags">` + `<span class="shelf-tag">IQ</span>` 등.

---

### 2.6 하단 CTA 버튼

| 상태 | 버튼 텍스트 |
|------|-------------|
| 목표 미선택 | **학습목표 고르기** |
| 목표 선택됨 | **추가 목표 고르기** |

- 현재 `btnSelectGoal` 문구 전환이 이미 구현되어 있음. "목표 다시 선택"으로 초기화할 때 다시 "학습목표 고르기"로 되돌리면 됨.

## 3. HTML 구조 보완 체크리스트

- [ ] **좌측 사이드바 영역**  
  - 연도 + 분기 + 선택 목표(제목, 설명) + "목표 다시 선택" 버튼을 하나의 컬럼으로 묶기.
- [ ] **인트로 문구**  
  - `intro-state--done` 내용을 이미지 문구로 변경하고, 목표명은 JS로 치환.
- [ ] **책 카드**  
  - 선택 시 표시할 부제/설명/태그용 요소가 있으면 추가 (예: `card-subtitle`, `card-tag`).  
  - 이미지처럼 "조직도가 리셋된다!" 같은 부제는 목표별 데이터로 넣을 수 있도록.
- [ ] **책장 하단 태그**  
  - `shelf-tags` + `shelf-tag` 추가.
- [ ] **사용자 정보**  
  - "지난 주 대비 44↑" 표시할 수 있도록 `rank-text` 내 마크업/데이터 확장.

---

## 4. CSS 가이드 (요약)

- **레이아웃**: `.myclass-wrap`을 grid 또는 flex로 **좌(사이드바) | 우(인트로+책장+CTA)** 2단 구성.
- **인트로**  
  - `intro-state--done`의 목표명: 보라색(`#7B6CF5` 등) + 굵게.  
  - D-day 문구: 주황색.
- **책 카드**  
  - `.is-selected .goal-card`: 청록 계열 배경.  
  - `.goal-item` (미선택): 카드별 파스텔 배경 클래스.
- **사이드바**  
  - 고정 너비(예: 240px) 또는 비율.  
  - "목표 다시 선택"은 왼쪽 화살표 아이콘 + 텍스트.

---

## 5. JavaScript 동작 정리

| 동작 | 현재 | 가이드 보완 |
|------|------|-------------|
| 목표 설정 | `applySelectedState(goalId)` | 좌측 사이드바에 선택 목표 제목/설명 채우기, 인트로 문구에 목표명 넣기 |
| 목표 해제 | 없음 | "목표 다시 선택" 클릭 시 `selectedGoalId = null`, 모든 카드/인트로/CTA 초기화, 사이드바 목표 영역 숨김 또는 비우기 |
| 인트로 문구 | `intro-state--done`만 표시 | done 상태일 때 문구를 "[목표명], 지금부터..." / "D-64... 나만의 지식을..." 로 설정 |
| CTA 버튼 | "추가 목표 고르기"로 변경됨 | 목표 해제 시 "학습목표 고르기"로 복원 |

---

## 6. 데이터 확장 제안

- **목표(goal)**  
  - `title`, `desc` 외에 선택 시 책장에 쓸 **부제**(예: "조직도가 리셋된다!"), **한 줄 설명**, **카테고리 태그**(IT 테크, 웰니스 등) 필드 추가 시 이미지와 동일한 표현 가능.
- **사용자**  
  - `rank`, `rankChange` (숫자), `rankDirection` (up/down) 등으로 "지난 주 대비 44↑" 조합.

---

## 7. 작업 순서 제안

1. 레이아웃: 좌측 사이드바 + 오른쪽(인트로·책장·CTA) 구조와 반응형 비율 잡기.
2. 인트로: 목표 선택 후 문구·색상 이미지와 동일하게 수정.
3. 사이드바: 연도/분기/선택 목표/"목표 다시 선택" 마크업과 표시·숨김 로직.
4. 책 카드: 선택 시 책 표지·배지·하트·제목·설명·태그 스타일 및 데이터 연동.
5. 책장 하단 태그 영역 추가.
6. 사용자 정보 바 "지난 주 대비" 표시.
7. "목표 다시 선택" 클릭 시 전체 상태 초기화 및 CTA 문구 복원.

이 가이드를 따라가면 제공된 이미지와 동일한 UI·UX로 `myclass_02.html`을 맞출 수 있습니다.  
특정 블록(예: 사이드바 마크업, 인트로 문구)부터 적용하고 싶으면 해당 섹션 번호를 알려주시면 됩니다.
